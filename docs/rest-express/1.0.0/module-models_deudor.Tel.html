Class: Tel

Tel()

Extrae y separa números telefónicos fijos (7 dígitos) y celulares (10 dígitos) a partir de una cadena de texto libre.

Constructor

new Tel()

Source:
  • models/deudor.js, line 1
Example
const tel = new Tel('3101234567 2345678');
console.log(tel.celular); // '3101234567'
console.log(tel.fijo);    // '2345678'
