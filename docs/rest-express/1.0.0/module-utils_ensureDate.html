Module: utils/ensureDate

Utilidades de conversión y formateo de fechas. Provee funciones para convertir de forma segura valores de fecha (strings, objetos Date) a objetos Date válidos, manejando timezone Colombia.

Methods

(static) ensureDate(dateInput) → {Date|null}

Convierte de forma segura un valor de fecha a un objeto Date válido. Maneja múltiples formatos de entrada y añade el offset de timezone de Colombia (-05:00) automáticamente cuando no se especifica zona horaria.

Reglas:

  • Input falsy → null
  • Ya es Date → valida y retorna (o null si es inválida)
  • String solo-fecha (2026-02-16) → Agrega T00:00:00-05:00
  • String fecha-hora sin timezone → Agrega -05:00
  • String con timezone → Parsea directamente

Example
ensureDate('2026-02-16');              // Date en Colombia TZ
ensureDate('2026-02-16T14:30:00');     // Date con offset -05:00
ensureDate(null);                      // null
ensureDate(new Date('invalid'));       // null

(static) formatDateToString(date) → {string}

Formatea un objeto Date a una cadena legible en español colombiano. Utiliza Intl.DateTimeFormat con timezone America/Bogota para garantizar consistencia independientemente del servidor/contenedor.
