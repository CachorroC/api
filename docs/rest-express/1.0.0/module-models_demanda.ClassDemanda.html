Namespace: ClassDemanda

models/demanda.ClassDemanda

Representa una demanda judicial dentro del sistema. Encapsula la lógica de transformación de datos crudos (RawDb) a una estructura tipada que incluye obligaciones, fechas procesales, medidas cautelares, notificaciones y montos económicos (capital adeudado, avalúo, liquidación).

console.log(demanda.capitalAdeudado); // Prisma.Decimal
