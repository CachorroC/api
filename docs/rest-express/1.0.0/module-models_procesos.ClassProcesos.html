Class: ClassProcesos

models/procesos.ClassProcesos()

Gestiona la colección de procesos judiciales asociados a una carpeta. Permite consultar la API de la Rama Judicial por llave de proceso, filtrar procesos privados, y sincronizar los resultados con la base de datos mediante operaciones connectOrCreate de Prisma.
